## デザインシステムとは？

デザインと実装を一貫性をもって提供するための仕組み。

## デザインシステムに含まれる要素は？

- ガイドライン
  - デザインに関する方針やルールを説明したもの
  - データの表示方法、ボタンの配置、色の使い方、アクセシビリティなどについて文章で説明する
- デザイントークン
  - デザインの要素のルールを定義したもの
  - 色、フォントサイズ、余白などの値を変数として定義する
- コンポーネントライブラリ
  - デザイントークンを利用して実装されたコンポーネントの集合体

## デザインシステムを作るメリットは？

- デザイン面
  - 企業、サービスのブランディングを確立することができる
  - サービスのデザインアイデンティティを統一することができる
  - ユーザー体験の一貫性を保つことができる
- 実装面
  - デザイナーとエンジニアのコミュニケーションコストを下げることができる
  - 社外のステークホルダーとのコミュニケーションコストも下げることができる
  - 開発の効率化ができる
