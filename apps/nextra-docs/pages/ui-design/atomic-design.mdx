## 化学要素になぞらえたデザインパターン

- Atoms
  - Button, Input, Label, Heading, Paragraph, etc.
- Molecules
  - Search Form, Navigation Bar, etc.
- Organisms
  - Header, Footer, etc.
- Templates
  - Home Page Template, About Page Template, etc.
- Pages
  - Home Page, About Page, etc.

### Atoms (原子)

- コンポーネントの最小単位
- ユーザーの目的を達成する「手段」を提供するが「機能」を持たない
  - ボタンは押すことができるが、ボタンを押すこと自体は機能ではない
- コンテキストを持たない
- 例
  - HTMLの基本要素
    - Button -> `<button>`
    - Label -> `<label>`
    - Text -> `<p>`
    - TextInput -> `<input type="text">`
    - etc.
  - デファクトスタンダードなUIコンポーネント
    - Icon
    - Badge
    - DatePicker, ColorPicker
    - Panel, Card
    - etc.

### Molecules (分子)

- Atomsの組み合わせにより構成されるコンポーネント
- ユーザーの目的を達成する「機能」を提供するが、何のための「機能」かは決めない
  - 検索フォームは検索する機能を提供するが、検索の対象（＝コンテキスト）は決めない
- コンテキストを持たない
- 例
  - SearchForm （検索フォーム）
    - TextInput
    - Button
  - NavBar （ナビゲーションバー）
    - Logo
    - Menu
  - Pagination （ページネーション）
    - Button
    - Text
  - Modal, Dialog （モーダル、ダイアログ）
    - Button
    - Text
    - etc.
  - etc.

### Organisms (有機体)

- Atoms、Moleculesの組み合わせにより構成されるコンポーネント
- ユーザーの特定の目的を達成する「機能」を提供する
  - ヘッダーは「Webサイト全体の」コンテンツナビゲーションの機能を提供する
  - ヘッダーの検索フォームは「Webサイト全体の」検索機能を提供する
- コンテキストを持つ
- 単独でコンテンツを提供している
- 例
  - Header （ヘッダー）
    - Logo
    - SearchForm
    - Menu
  - Footer （フッター）
    - Logo
    - Menu
    - etc.
  - etc.

### Templates (テンプレート)

- Pageの雛形
- Pageのレイアウト構造を定義する
- コンポーネントの粒度に関わるレイヤーは、Atoms、Molecules、Organismsまで

### Pages (ページ)

- 具体的なコンテンツ（テキスト、画像、動画、etc.）が入った画面
